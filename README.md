# IBM Data Science Capstone Project

## Overview
This repository contains my completed IBM Data Science Capstone Project. The project demonstrates the application of SQL, data analysis, data wrangling, and visualization techniques learned throughout the IBM Data Science Professional Certificate program.

The capstone includes hands-on labs, Jupyter Notebooks, SQL queries, exploratory data analysis, and a final presentation summarizing the results and insights from the completed modules.

---

## Project Objectives
- Practice SQL queries and database operations
- Perform data wrangling and cleaning
- Analyze datasets using Python and Pandas
- Create visualizations and dashboards
- Understand how SQL integrates into the data science workflow
- Present findings in a professional presentation format
